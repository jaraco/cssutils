what is it
----------
A Python package to parse and build CSS Cascading Style Sheets. (Not a renderer though!)

about this release
------------------
0.9.6b1 is a bugfix release.


main changes
------------
    + BUGFIX: Fixed ``CSSPageRule.selectorText = ''`` which does reset the selector now
    + BUGFIX (minor): Removed false references in a few modules' ``__all__`` list

    - IMPROVEMENT: Jython 2.5 (from RC4) passes all tests now
