what is it
----------
A Python package to parse and build CSS Cascading Style Sheets. (Not a renderer though!)

about this release
------------------
0.9.6 is a full release.

main changes
------------
+ BUGFIX: Definition of macro for `positivenum` in cssutils profiles actually did allow nagative numbers, fixed (thanks to Jason R. Coombs)

download
--------
For download options see http://cthedot.de/cssutils/

cssutils needs Python 2.4 or higher or Jython 2.5 and higher (tested with Python 2.6.3, 2.5.2, 2.4.4 and Jython 2.5.1 on Vista only)
